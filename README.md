# Immediac CEO Dashboard

A lightweight CEO dashboard for tracking KPIs, initiatives, projects, to-dos, and delegation.

**Stack:** Pure HTML / CSS / JS — no frameworks, no build tools.

## Structure

```
ceo-dashboard/
├── index.html
├── assets/
│   ├── css/
│   │   ├── reset.css       — normalize browser defaults
│   │   ├── tokens.css      — design tokens (colours, spacing, type)
│   │   ├── layout.css      — page structure and grid
│   │   └── components.css  — cards, accordions, badges, tables
│   └── js/
│       ├── data.js         — mock data (replace with API later)
│       ├── accordion.js    — accordion + todo interaction
│       └── main.js         — render functions
└── README.md
```

## Running locally

Just open `index.html` in a browser — no server needed.
